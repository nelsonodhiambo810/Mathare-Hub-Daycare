# 🧸 Mathare Hub Daycare Management System

A native Android application built to streamline daycare operations, attendance tracking, and parent communication. 

This prototype was developed to replace manual paper logs with a sleek, automated digital system, allowing teachers to focus less on paperwork and more on the children.

## ✨ Key Features
* **Secure Admin Dashboard:** A quick-glance view of daily attendance and incident metrics.
* **Attendance & Meal Tracking:** Fast, intuitive toggles to record daily check-ins and feeding schedules.
* **The "Ouch Log" (Incident Reporting):** A dedicated system to log medical incidents and first-aid actions.
* **Automated Parent Reporting:** Generates pre-formatted daily summaries sent directly via WhatsApp or SMS using Android Intents.
* **Modern UI/UX:** Built entirely with Jetpack Compose featuring a custom dark-mode theme, expandable contact cards, and smooth animations.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material Design 3)
* **Architecture:** MVVM (Model-View-ViewModel)
* **State Management:** Kotlin Coroutines & StateFlow
